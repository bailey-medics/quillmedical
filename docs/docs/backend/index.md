# Backend Python API

## Overview

The Quill Medical backend is built with FastAPI and provides:

- RESTful API for patient records, messaging, and letters
- Authentication and authorization with JWT tokens
- FHIR integration via HAPI FHIR server
- Push notification support
- PostgreSQL database with SQLAlchemy ORM
